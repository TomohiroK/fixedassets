# ASEAN Fixed Asset Depreciation Rules - Overview

## Supported Countries

| Code | Country | Currency | System | Methods |
|------|---------|----------|--------|---------|
| JP | Japan | JPY | Depreciation | SL, 200%DB (guarantee amount) |
| SG | Singapore | SGD | Capital Allowance | SL only (IA+AA) |
| TH | Thailand | THB | Depreciation | SL, DB, SYD |
| MY | Malaysia | MYR | Capital Allowance | SL only (IA+AA) |
| VN | Vietnam | VND | Depreciation | SL, DB, Units of Production |
| ID | Indonesia | IDR | Depreciation | SL, DB (4 groups) |
| PH | Philippines | PHP | Depreciation | SL, DB, SYD |
| MM | Myanmar | MMK | Depreciation | SL only (prescribed rates) |
| KH | Cambodia | KHR/USD | Depreciation | SL (class 1), DB (classes 2-4, pooled) |
| LA | Laos | LAK | Depreciation | SL, DDB, Units of Production |

## Key Differences

### Salvage Value
- **JP**: 1 JPY (memorandum value)
- **SG, MY**: N/A (capital allowance system, balancing adjustments on disposal)
- **All others**: Effectively 0 (no mandatory minimum)

### Non-Depreciable Assets
- **All countries**: Land is not depreciated
- **SG**: Buildings are not qualifying assets for capital allowances (except Industrial Building Allowance)
- **KH**: Construction in progress not depreciated until completion

### Implementation Priority
1. JP - Complex (guarantee amount, revised rate)
2. SG - Different system (Capital Allowance with IA+AA)
3. TH - Standard depreciation (with vehicle cost cap)
4. MY - Different system (Capital Allowance with IA+AA)
5. ID - Standard but 4-group classification system
6. VN - Standard with min-max useful life ranges
7. PH - Most flexible (taxpayer determines useful life)
8. MM - Simple (SL only, prescribed rates)
9. KH - Unique (pooled declining balance for classes 2-4)
10. LA - Standard with prescribed rates
