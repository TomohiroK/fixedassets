# Philippines (PH) - Fixed Asset Depreciation Rules

## Currency: PHP

## Depreciation Methods
- **Straight-line** (most common)
- **Declining balance**
- **Sum-of-years-digits**
- Any reasonable method is acceptable
- Change of method requires **BIR approval**

## Useful Life
- **No prescribed useful life** in tax code
- Taxpayer determines useful life based on professional judgment
- Common practice:

| Asset Category | Commonly Used Life |
|----------------|-------------------|
| Buildings | 20-30 years |
| Vehicles | 5 years |
| Machinery & equipment | 5-10 years |
| Furniture & fixtures | 5-10 years |
| Computers / electronics | 3-5 years |
| Office equipment | 5 years |
| Software | 3-5 years |
| Patents | Legal life |
| Goodwill | 5-10 years |

## Salvage Value
- **No mandatory minimum**. Taxpayer sets based on reasonable estimate.
- In practice, many use 0 or a nominal amount.

## Straight-Line
```
Annual Expense = (Cost - Salvage Value) / Useful Life
```

## Declining Balance
```
Annual Expense = Book Value x (SL Rate x multiplier)
```

## Special Rules
- Depreciation must be based on **acquisition cost only** (not revalued amount)
- Change of useful life requires BIR approval
- Once agreed with BIR, rate is binding unless new facts emerge
- Tax depreciation should generally conform to book depreciation

## Category Restrictions
- Most flexible system among ASEAN countries
- All methods available for most asset types
- Land is non-depreciable
- Construction in progress: not depreciated until completion

## Implementation Notes
- Since there are no prescribed rates, the system should allow full user control
- Default useful lives can be suggested but user should override freely
- Both SL and DB methods should be available for all depreciable assets
